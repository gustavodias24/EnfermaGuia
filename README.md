## Sobre o Projeto

Painel de Admin.
Cadastro de Hospitais.
Sistema de Login.
Sistema de Cadastro de Guias de Enfermagem.
Sistema de Cadastros de Sugestões.
Gerador de PDF ( dos Guias de Enfermagem ).
Muito mais....

### Tecnologias Utilizadas

- Java
- Android Studio
- Firebase
